在Anaconda环境下安装Tensorflow

2020-12-29
Tensorflow > 毕业设计

安装Anaconda

下载Anaconda

个人版Individual Edition。如果下载速度慢，可以复制下载链接到迅雷或者在清华大学开源镜像站TUNA中找合适的版本。

注意在安装过程中的“Advanced Options”有两个选项，推荐勾选第二项：“Register Anaconda as my default Python”。

添加环境变量

添加Anaconda安装根目录.\和.\Scripts\到系统变量。例如

C:\Users\MyUsername\anaconda3
C:\Users\MyUsername\anaconda3\Scripts

激活环境

在常用的终端——例如PowerShell——输入conda activate激活环境，使用conda deactivate退出环境。

conda是一个提供包管理与环境管理的工具，包管理与pip类似，环境管理则允许用户安装不同版本的python环境并可以快速切换。Anaconda是一个开源的Python发行版本，包含了conda、Python以及Python的库numpy、pandas等，就是个Python全家桶。1

为Anaconda添加国内镜像源2

TUNA提供了Anaconda仓库的镜像，运行以下指令可以添加：

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --set show_channel_urls yes

或者也可以直接修改个人文件夹下conda的配置文件.condarc为以下内容：

channels:
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
  - https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
show_channel_urls: true

安装Tensorflow

创建一个Tensorflow环境

conda create -n 环境名称 python=Python版本号
conda cretae -n tensorflow python=3.8.5

激活Tensorflow环境

conda activate tensorflow

安装Tensorflow

在刚才创建的名为tensorsflow的环境中安装tensorflow。

pip install tensorflow

如果下载速度很慢，可以上Links for tensorflow直接下载。然后确保安装了wheel，

pip install wheel

就可以直接使用如下指令进行安装

pip install "D:\Downloads\迅雷下载\tensorflow-2.4.0-cp38-cp38-win_amd64.whl"

安装成功之后，在tensorflow环境下进入python解释器，输入import tensorflow as tf，没有异常，表示安装成功。注意，Pyhton的深度学习软件包tensorflow是在Anaconda名为tensorflow的环境下安装的，在基础环境base中是没有tensorflow的。

退出tensorflow环境

conda deactivate
